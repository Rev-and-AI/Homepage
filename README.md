# 命名ルール
- 変数名等はケバブケース(aaa-bbb-ccc)で命名
- 記号などで省略せずに意味のある変数名をつける(あくまで例として: 運動方程式"F=ma"は"Force = mass * acceleration"とわかりやすく書くこと)
